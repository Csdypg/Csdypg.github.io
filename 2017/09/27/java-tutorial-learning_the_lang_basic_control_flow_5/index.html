return 语句

最后一个分支语句是return语句。return语句存在与当前的方法中，控制流程返回到方法被调用的地方。return语句有两种形式：一种返回一个值，一种不返回。返回值的时候只需要把返回的值（或者是用来计算返回值的表达式）放在return关键字之后。

返回值的数据类型必须有方法声明的返回值的数据类型匹配。当方法的声明为void时，需要使用不带返回值的return形式。
