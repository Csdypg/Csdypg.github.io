for 语句

for语句为迭代一个范围内的值提供了一种简介的方法。开发者经常称之为’for 循环’,因为它重复循环直到满足一个特定条件的方式。for循环的一般形式展示如下:

当你使用这种版本的for语句时，注意以下几点:

  • 初始化表达式初始化整个循环；它只随着循环开始执行一次.
  • 当终止条件的表达式球的false时，循环终止。
  • 增加表达式在循环的每一次迭代之后被调用，表达式的只增加或减少都时支持的。
