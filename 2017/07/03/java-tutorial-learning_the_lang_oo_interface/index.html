什么是Interface接口？

就像你之前学到的，对象通过暴露的方法定义与外界的交互。方法构成了对象与外部世界的interface接口；你电视机前面的按钮，是你和电视塑料盒子背后电子线路的交互接口。当按下“电源”按钮来打开和关闭电视机。

实现一个接口可以使一个类对于其应该提供的行为方法更加规范。接口规定了类与外部世界交互的约定，并且这个约定在编译器是强制执行的。如果你的类声明实现了某一个接口，所有在接口中定义的方法都必须出现在这个类的源代码中，否则就无法编译成功。

注意:

事实上，为了成功编译上述ACMEBicycle类，你必须在所有的实现方法添加public关键字。原因将在随后的课程类与对象，接口与继承中讲到。
